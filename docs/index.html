Unofficial Sunflower-Land Android App

An open-source, community-built Android App for Sunflower Land

About

The Unofficial Android App is open-source Android client for the blockchain based farming game Sunflower Land. Enjoy a native mobile experience, real-time notifications, and more—right from your phone!

Features

  • Full Sunflower Land gameplay
  • Background notifications for almost all categories (Some categories are WIP)
  • Built in Android Game Acceleration
  • Modern, responsive Android UI
  • Customizable settings and logs
  • Completely open-source and community maintained

A Note For Testers

  • The app can operate without the native browser activity. You can disable this in settings by turning on the “notifications only” toggle.
  • Clicking any notification (not the drop down arrow) will automatically open the app. But this can also be customized to open any app you desire.
  • To view settings:
    1. While the game is open, triple tap with 3 fingers
    2. Long press the app → App info → “Configure in Sunflower Land” or “Settings in Sunflower Land” (label may vary by phone)

How to Enable Push Notifications

  1. Insert your Farm ID into app settings (find it in-game: Settings → 3 dots → top of the game options panel → click to copy).
  2. Insert your Farm API key into app settings (find it in-game: Settings → 3 dots → General → API Key → Copy).
  3. Click the “Start Worker” button in settings, or simply open the app if you use the web-browsing feature (auto-start can be disabled in settings).

How to Install

  1. Download from The Google Play Store link above.

  2. Download the latest APK directly from GitHub: Latest Release

    The APK is named sunflower-land-unofficial-v[version].apk

For advanced users

You can build the app from source using Android Studio or the provided Gradle scripts.

  1. Clone or download this repository to your computer.
  2. Install the Android SDK Platform Tools (Android Studio is optional, but you must have the SDK and platform tools to build and install to Android devices).
  3. Open a terminal in the project root directory.
  4. Navigate to the mobile/android folder.
  5. Enable "Install from Unknown Sources" on your Android device if installing manually (usually in Settings > Security).
  6. Connect your Android device via USB and ensure USB debugging is enabled (see how to enable USB debugging).
  7. Run .\gradlew.bat installdebug (Windows) or ./gradlew installDebug (Linux/macOS) to build and install the debug APK directly to your connected device.
  8. If you prefer, you can still find the APK in app/build/outputs/apk/debug/app-debug.apk and install it manually.

For more details or troubleshooting, see the README or open an issue on GitHub.
